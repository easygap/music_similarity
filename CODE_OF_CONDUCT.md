# 행동 강령

이 프로젝트는 누구나 편하게 기여하고 사용할 수 있는 공간을 지향합니다.
이슈/PR/디스커션 어디든 아래 원칙을 지켜주세요.

## 기대하는 행동

- 다른 사람을 존중하고, 의견 차이는 데이터/근거로 풀어내기.
- 초보자의 질문도 그대로 환영하기. 가르치는 어조보다 같이 푸는 어조로.
- 코드/디자인 비판은 사람이 아니라 결과물에 대해서 하기.
- 문화/배경/성별/연령/장애 여부와 무관하게 동등한 기회를 제공하기.

## 금지되는 행동

- 차별/혐오 발언, 인격 모독, 비하성 농담.
- 성적 메시지 또는 그에 준하는 시각자료.
- 개인정보 무단 공개 (실명/주소/연락처/사진 등).
- 의도적 트롤링 / 욕설 / 위협.

## 강제 / 신고

행동 강령 위반을 목격하거나 직접 겪은 경우, GitHub Security Advisory 또는
저장소 소유자에게 비공개로 알려주세요. 다음 절차로 처리됩니다.

1. 접수 후 영업일 기준 72시간 이내에 회신.
2. 사실 확인을 위해 추가 정보 요청 가능.
3. 경중에 따라 경고 → 일시 차단 → 영구 차단 단계로 조치.

## 적용 범위

이 문서는 본 저장소의 이슈, PR, 디스커션, 커밋 메시지 같은 공개 채널뿐 아니라
프로젝트와 연결된 외부 커뮤니티(트위터/X 등) 에서의 활동에도 적용됩니다.
