이용 약관

최종 업데이트: 2026-06-01

본 약관은 SoundMatch (이하 "서비스") 의 이용 조건을 정합니다. 서비스에 접속하거나 사용하는 것은 본 약관에 동의하는 것으로 간주됩니다.

1. 서비스의 목적

서비스는 사용자가 업로드한 음원 또는 카탈로그 곡을 librosa 와 scikit-learn 으로 분석해 유사도, 메타 지표, 추천 목록, 비교 결과, 즐겨찾기, 공유 링크, 내보내기 파일을 제공하는 도구입니다. 학술/취미 목적의 참고용 으로 제공되며, 분석 결과는 저작권, 표절, 라이선스, 계약, 광고 심의, 상업적 유사성에 대한 법적 판단 근거가 될 수 없습니다.

2. 사용자 책임

  • 본인이 권리를 보유하고 있거나, 분석에 사용해도 무방한 음원만 업로드해야 합니다.
  • 저작권/초상권 등 타인의 권리를 침해하는 파일을 업로드해서는 안 됩니다.
  • 분석 결과, 공유 링크, 내보내기 파일을 저장하거나 제3자에게 전달할 때의 책임은 사용자에게 있습니다.
  • 분석 결과를 저작권 분쟁, 라이선스 판단, 법적 판단, 상업적 의사결정의 단독 근거로 사용해서는 안 됩니다.
  • 악성 코드, 자동화된 대량 요청, Rate limit 우회, 비정상적으로 큰 클라이언트 오류 비콘, API 남용, 보안 취약점 탐색은 금지됩니다.

3. 데이터와 결과물의 범위

업로드한 음원은 분석을 위해 일시적으로 처리되며, 서비스가 영구 보관하지 않습니다. 최근 분석 결과, 카탈로그 즐겨찾기, 검색/정렬 선호, 최근 본 곡 같은 브라우저 편의 기능은 사용자의 브라우저 localStorage 에 저장됩니다.

공유 링크는 분석 결과 일부를 URL hash 에 담아 복원하며, JSON/CSV/SVG/PNG 내보내기 파일은 사용자가 직접 저장하고 공유하는 결과물입니다. 서비스 안정성 개선을 위해 클라이언트 오류 비콘이 전송될 수 있으며, 자세한 저장/전송 범위는 개인정보 처리방침을 확인해주세요.

4. 라이선스 / 저작권

서비스의 소스 코드는 GitHub easygap/music_similarity 에서 MIT 라이선스 하에 공개되어 있습니다. 사용자가 업로드하는 음원의 저작권은 해당 권리자에게 있으며, 서비스는 일시적인 분석 외 어떤 권리도 갖지 않습니다.

5. 서비스 변경 / 중단

서비스는 사전 공지 없이 운영을 중단하거나 API 사양, 기능, 운영 한도를 변경할 수 있습니다. 변경 사항이 큰 경우 GitHub Releases 와 CHANGELOG.md 를 통해 안내합니다.

6. 면책

서비스는 "있는 그대로(AS-IS)" 제공됩니다. 운영자는 서비스의 정확성, 가용성, 특정 목적 적합성에 대해 어떠한 보증도 하지 않으며, 서비스 이용으로 발생한 직·간접적 손해에 대해 책임지지 않습니다.

7. 분쟁 해결

본 약관과 서비스 이용에 관한 분쟁은 대한민국 법령을 준거법으로 하고, 관할 법원은 운영자 주소지 관할 법원으로 합니다.
