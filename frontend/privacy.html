개인정보 처리 방침

최종 업데이트: 2026-05-13

SoundMatch (이하 "서비스") 는 사용자의 음악 파일을 받아 유사도 분석을 제공하는 도구입니다. 본 문서는 서비스 운영 과정에서 어떤 정보를 받고, 어떻게 처리하는지 투명하게 안내하기 위해 작성되었습니다.

1. 수집하는 정보

  • 업로드된 음원 파일: 분석을 위해 일시적으로 서버 디스크에 저장합니다.
  • 요청 메타데이터: 요청 ID, 접속 IP, 사용된 브라우저(User-Agent), 응답 시간 등 운영 로그에 남는 정보.
  • 분석 결과: 응답으로 사용자에게 즉시 반환되며 서버에 보관되지 않습니다.

2. 보관 기간

업로드된 음원은 분석이 끝나는 즉시 서버에서 자동 삭제됩니다. 운영 로그(요청 ID, 처리 시간 등)는 장애 분석을 위해 최대 30일까지 유지되며, 그 이후 자동 폐기됩니다.

3. 이용 목적

  • 업로드된 음원에 대한 유사도 분석 결과 제공
  • 서비스 안정성/성능 모니터링 및 장애 대응
  • 이상 트래픽/남용 차단 (Rate limit 등)

4. 제3자 제공 / 학습 사용

업로드된 음원은 제3자에게 제공되지 않으며, 어떤 형태의 머신러닝 모델 학습이나 카탈로그 확장에도 사용되지 않습니다. 유사도 비교에 쓰이는 카탈로그(data/dataset.csv)는 사전 추출된 특성값으로만 구성되어 있고, 사용자 업로드와는 무관합니다.

5. 쿠키 / 로컬 저장소

서비스는 별도의 추적용 쿠키를 사용하지 않습니다. 다만 사용자 편의를 위해 다음 항목을 브라우저의 localStorage 에 저장합니다:

  • 선택한 테마(다크/라이트), 언어 (ko/en)
  • 최근 분석 결과 5건 (히스토리 사이드바에 사용)

이 정보는 사용자의 브라우저 안에만 존재하며 서버로 전송되지 않습니다. 브라우저의 사이트 데이터를 지우면 모두 삭제됩니다.

6. 권리 행사 / 문의

서비스 이용 과정에서 추가 문의 사항이 있다면 GitHub Issues 또는 보안 관련 사안의 경우 Security Advisory 로 비공개 신고해 주세요.

7. 변경 이력

방침이 바뀌면 본 페이지의 "최종 업데이트" 날짜를 갱신하고, 중요한 변경의 경우 메인 화면에 공지합니다.
